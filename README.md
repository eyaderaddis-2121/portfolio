# Eyaderaddis Beleta Azazh - Full Stack Developer Portfolio

A premium, professional, and highly responsive personal portfolio website built with React, Node.js, Express, and SQLite.

## Features

- **Modern UI/UX**: Clean, elegant design with glassmorphism and smooth animations.
- **Full Stack**: Integrated with a Node.js/Express backend and SQLite database.
- **Dynamic Content**: Projects and testimonials are fetched from the backend API.
- **Contact System**: Fully functional contact form that saves messages to the database.
- **Responsive Design**: Optimized for mobile, tablet, and desktop devices.
- **Dark/Light Mode**: User-friendly theme toggle with persistent state.
- **SEO Optimized**: Built with performance and accessibility in mind.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion, Lucide React.
- **Backend**: Node.js, Express.
- **Database**: SQLite (using `better-sqlite3`).
- **Styling**: Tailwind CSS (Primary), Bulma (Optional).

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your configurations (see `.env.example`).

4. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

### Frontend (Vercel)
- Connect your GitHub repository to Vercel.
- Set the build command to `npm run build`.
- Set the output directory to `dist`.

### Backend (Render/Railway)
- Connect your repository to Render or Railway.
- Set the start command to `node server.ts` (ensure you have `tsx` or compile to JS).
- Configure your database environment variables.
